# FindHoliday v2.1

## Основное

**50 встроенных праздников** на 5 языках.

**Свои праздники** - добавляйте сколько угодно.

**Поиск** - быстрый поиск по названию.

**Избранное** - закрепляйте нужные.

**5 языков** - РУ, EN, DE, FR, UK.

## Использование

1. Откройте `index.html`
2. Ищите праздники или добавляйте свои
3. Все сохраняется в браузере
4. Смените язык если нужно

## Запуск Сервера

```
python -m http.server 8000
```

Потом откройте `http://localhost:8000`

## Сохранение Данных

Все в браузере (режим гостя):
- Язык
- Избранное
- Свои праздники

Никаких аккаунтов.
